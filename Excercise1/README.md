# Exercise 1: Setting Up Ubuntu 20.04 with Vagrant

## 📌 Task

Set up Ubuntu 20.04 LTS using Vagrant with a private network set to DHCP.

## ✅ Steps Taken

1. Installed Vagrant and VirtualBox.
2. Created and customized the `Vagrantfile` with DHCP private network.
3. Ran `vagrant up` to start the virtual machine.
4. Ran `ifconfig` inside the VM and took a screenshot of the output.
