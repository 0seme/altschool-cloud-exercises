# AltSchool Cloud Exercises

Linux and virtualization exercises from AltSchool's Cloud Engineering track.

## Exercises

### [Exercise 1 — Setting Up Ubuntu 20.04 with Vagrant](./Excercise1)

Provisioned an Ubuntu 20.04 VM with Vagrant, configured with a private network on
DHCP, and verified the network config with `ifconfig`.

### [Exercise 2 — 10 Linux Commands](./Excercise2)

A walkthrough of ten Linux commands (`w`, `id`, `cal`, `less`, `bg`, `dig`, `du -sh`,
`du -ah`, `history`, `sudo`) with example output for each.

## Tech Stack

- Vagrant + VirtualBox
- Ubuntu 20.04 LTS
- Core Linux command-line tools
